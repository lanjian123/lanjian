clear : 清除屏幕；
pwd : 查看当前路径；
ls : 查看所有文件！
mkdir : 创建文件夹！
git : git init 创建仓库！
ls : ls -a 查看所有当前所有文件（包含隐藏文件）
touch ：touch readme.text 创建文件
git:配置 Git is a version control system. Git is free software.
vim ： 在git 的CMD里面写
git add : 告诉仓库添加一个版本
git add . : 提交所有文件到暂存区
git commit -m : 提交到版本（m 用来注释提交的内容！）
git diff : 查看被修改的内容记录
git status : 查看工作区和暂存区里面是否有需要提交的文件{ 红色字体代表我们文件在工作区； 绿色代表我们文件在暂存区； nothing commit 代表文件已经全部提交到git仓库 }
git log : 查看修改提交的日志（记录）
git reset --hard HEAD^{回退到上一个版本}{如果想回退到上上一个版本需要加两个^,以此类推}
git reset --hard xxx : 可以通过commit ID值回退到对应的版本
git reflog ： 查看commit id 值
rm : rm file.txt(删除指定文件！)；
git checkout -- file1.txt : 撤销修改！（只能修改保存到库里的撤销！）
cd desktop :
git push -u origin master:提到运程！
git push -u origin master 把代码提交到远程仓库的主分支。从远程仓库拉取最新内容；
git pull 更新本地仓库的内容 从远程仓库拉取最新内容
。gitgnore : 上传远程仓库，忽略的文件
git branch : 查看分支！
git branch lsj: 创建分支！ /************************************************/
